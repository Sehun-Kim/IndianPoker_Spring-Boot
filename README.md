# IndianPoker Web

### Environment
- JAVA 8
- Spring-Boot 2.1.1
- Gradle
- Spring Data JPA
- Hibernate
- H2 DataBase(Temporary)
- Handlebars
- WebSocket(Sock.js)

---

### Requirement

1. 사용자 관리
    - 로그인
    - 프로필 사진 등록
2. IndianPoker Game
    - 두 명의 Player가 동시에 게임을 실행
    - 게임 결과 DB에 사용자별로 등록
3. 사용자 랭킹 조회
