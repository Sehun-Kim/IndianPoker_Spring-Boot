# IndianPoker Web

### Environment
- JAVA 8
- Spring-Boot 2.1.1
- Gradle
- Spring Data JPA
- Hibernate
- H2 DataBase(Temporary)
- Handlebars

---

### Requirement

1. 사용자 관리
    - 로그인(추후에 Spring-Security 쓸 것)
    - 프로필 사진 등록
2. IndianPoker Game
    - auto gamer와 게임
    - 게임 결과 DB에 사용자별로 등록
3. 사용자 랭킹 조회
